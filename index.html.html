Cleanup Options

What Each Option Cleans:

Cookies

Small text files websites store to remember your preferences, login status, and tracking data.

Cache

Temporary files (images, CSS, JS) stored to speed up page loading on subsequent visits.

Local Storage

Website data stored in your browser that persists between sessions (settings, preferences).

Browsing History

Record of websites you've visited, used for autocomplete and suggestions.

Note: This is a simulation for educational purposes. Actual browser cleanup requires using your browser's settings.
